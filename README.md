# Kontrola uzgona plastičnih okana

Aplikacija za proračun stabilnosti plastičnih okana na uzgon s grafičkim prikazom presjeka.

## 🚀 Funkcionalnosti

- Proračun faktora sigurnosti na uzgon (FS)
- Automatski izračun potrebne visine betonskog prstena
- Razdvajanje:
  - suhog i uronjenog tla
  - zone prstena i proširenja
- Grafički prikaz:
  - temeljne ploče
  - okna
  - betonskog prstena
  - tla (šrafura)
  - razine vode
  - uzgona i težina

---

## 📥 Ulazni podaci

- Kota terena
- Kota nivelete
- Razina podzemne vode
- Promjer okna (D)
- Dimenzija temeljne ploče (a)
- Debljina temeljne ploče (t)
- Debljina prstena (b)
- Faktor sigurnosti (FS)
